Google Cloud Setup — one-time per deployment
Your video never leaves your machine. Only 200px-wide thumbnail frames are uploaded to your team's Google Drive. Completely free to run.
STEP 01
Create Google Cloud Project
Go to console.cloud.google.com → New Project. Name it vfx-tracker.
STEP 02
Enable APIs
APIs & Services → Library → enable Google Drive API and Google Sheets API.
STEP 03
Create OAuth Client ID
Credentials → Create OAuth Client ID → Web Application. Add your hosting URL to Authorized JavaScript Origins. For local testing add http://localhost:8080.
STEP 04
Paste Client ID below
Copy the ID ending in .apps.googleusercontent.com. To pre-fill for your whole team, put it in HARDCODED_CLIENT_ID in the script and redeploy.
Saved in your browser. Each team member pastes theirs once on first use.
